# Security Policy

## Reporting a Vulnerability

Please do not open a public issue for security-sensitive reports.

Use GitHub Security Advisories when the repository is public:

- `https://github.com/canmutioglu/firehose-mcp/security/advisories/new`

If GitHub private reporting is not available yet, contact the maintainer privately through the repository owner profile before disclosing details publicly.

## Supported Versions

Security fixes will be applied to the latest published release line.
